Rat :

On mesure le poids de plusieurs groupes de rats soumis à des régimes différents. Un régime est caractérisé par le type et la quantité de nourriture.

Données :
   Col 1 = rat weight gain (in grams)
   Col 2 = diet amount (1 = high, 2 = low)
   Col 3 = diet type   (1 = beef, 2 = pork, 3 = cereal)

Questions :
Déterminer s'il existe une différence entre les groupes, et identifier l'influencede chacune des grandeurs.
