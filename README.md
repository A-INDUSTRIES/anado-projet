# Project de Machine Learning

## Environnement de travail

Utilisation de `uv` pour gérer les packages python. \
Pour commencer le développement:
```bash
git clone https://github.com/A-INDUSTRIES/machlearn-projet.git
cd machlearn-projet
uv sync 
source .venv/bin/activate
# ou bien .venv/Scripts/activate sur Windows
```

## Structure du projet
Un fichier .ipynb est attribué par méthode vue au cours. \
Les consignes du travail ainsi que les aides fournies sont présentes dans le dossier consignes-et-aides.
Les données du problème (autrement dit l'input) sont dans le dossier data.
Tous les résultats se trouvent dans le dossier results.
